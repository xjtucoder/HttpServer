## Oh-Server

一个用C++写的简单的epoll多线程http服务器。

`httpserver.cpp`                      主函数文件

`locker.cpp`和`locker.h`              互斥锁、信号量和条件变量的封装类

`threadpool.h`                        线程池类模板

`http_parser.cpp`和`http_parser.h`    解析http请求的类

`http_process.cpp`和`http_process.h`  处理http请求的类



##  usage

`make`

`./Oh-Server <port>`
